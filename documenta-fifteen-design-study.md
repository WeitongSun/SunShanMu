# documenta fifteen — 设计思路与风格研究

网站：https://documenta-fifteen.de/en/
策展方：ruangrupa（印尼雅加达艺术家集体）
核心概念：lumbung（印尼语"粮仓"，代表集体储存与共享资源）

---

## 一、设计哲学：视觉形式即概念表达

documenta fifteen 的视觉设计不是"展览配套"——它**本身就是作品的一部分**。

策展方 ruangrupa 将 *lumbung* 的原则——集体性、共享、去中心化——直接写入了设计语言：没有一个单一的"主导风格"，而是多个系统并置、互相干扰，形成一个生态。整个视觉体系体现的是**非等级制度下的多元共存**，而不是一个品牌统治所有元素。

这种思路体现在每一个具体设计决策中：字体系统里的"破格"、颜色的平权并置、无阴影无装饰的界面、把无障碍访问作为设计核心而非功能附加。

---

## 二、色彩系统

只有四种颜色，且都具有同等的视觉权重——没有主色、副色、点缀色之分：

| 名称 | 色值 | 气质 |
|------|------|------|
| 绿 Green | `rgb(41, 155, 112)` / `#29 9B 70` | 有机、生长、热带 |
| 红 Red | `rgb(245, 82, 46)` / `#F5 52 2E` | 紧迫、温热、政治性 |
| 黄 Yellow | `rgb(255, 221, 20)` / `#FF DD 14` | 活力、民间节庆 |
| 紫 Purple | `rgb(186, 82, 216)` / `#BA 52 D8` | 神秘、反传统 |

**使用规则：**
- 全页面按**色块分区**（full-bleed sections），每个模块是一整块纯色
- 同一页面中颜色交替出现，无渐变、无混合、无透明度
- 文字永远是黑色 `rgb(0,0,0)`，确保在任意颜色背景上的可读性
- 页脚使用紫色，创造一种"出戏"的收尾感
- 完全没有白色背景——白色只出现在艺术家图片的白边上

**为什么这样有效：**
四种颜色都是高饱和度、中明度——它们在色相上差异巨大，不会产生视觉层级，彼此平等。这在视觉上实践了"去中心化"：没有哪个板块比另一个更"重要"。

## 三、字体系统：核心在 MBI-DCMNT15

这是整个设计中最有力、最原创的部分。

### MBI-DCMNT15（定制展示字体）

这是专为 documenta fifteen 定制的字体，其核心机制是**在同一个字体文件中包含两套完全不同风格的字形**：

- **几何粗体字形**：A、B、C、D、E、M、S、T、U 等大写字母——厚重、方正、具有海报感，近似粗体无衬线或重型黑体
- **手写/草书字形**：部分字母（如 n、d、g）使用流动的、像手写或草书的形态，完全不与几何字母"配套"

当这两类字形在同一个单词中同时出现，产生了一种**单词内部的张力**——文字在阅读过程中发生"风格破裂"，读者需要重新辨认。

**视觉示例（截图中可见）：**
- `VENUES` → "N" 变成了流动的草书 n，读作 "VE**~~nn~~**UES"（真实效果是中间那几个字母变成行书）
- `MEDIA LIBRARY` → "d" 变成了一个手写体小写 d
- `LUMBUNG MEMBERS & ARTISTS` → 某些字母带有有机弯曲的笔触

这种设计意图非常明确：**大写/机构/纪念碑式的字体，被手写/个人/身体性的笔触"入侵"**。这是 lumbung 精神的字体化——集体中保有个体的声音。

**技术规格：**
- font-weight: 400（字重并不重，视觉冲击来自字形本身）
- font-size: 133px+ （hero 区域），自动填满容器宽度（`text-fit-distribution` 类）
- 全大写使用，无 letter-spacing 修改

### Editorial Regular / Light / Italic（正文衬线字体）

法国出版风格衬线字体，用于：
- 正文段落（42px，行高 79px——几乎是两倍行距，非常宽松）
- 艺术家姓名列表（H3，42px）
- 表现一种"严肃出版物"的质感，与标题字体的躁动形成对比

### Söhne Regular（界面/功能字体）

Klim 字库，略带人文主义气息的无衬线字体，分类为 `monospace`（但实际并非等宽）。用于：
- 导航链接（29px）
- 按钮文字（29px）
- 功能性文字（日期、标签等）
- 在视觉上"退场"，不与展示字体争夺注意力

### Gudea Regular / Guggeli Regular（ticker 字体）

用于页面顶部的两条滚动字幕（ticker），两条 ticker 使用不同字体，强化了"不同频道/声音"的感觉。字号与导航相同（29px）。

## 四、页面结构与布局逻辑

### 首页模块顺序（从上到下）

```
[导航栏]        透明背景，Menu + 无障碍工具栏 + 语言
[Hero]          绿色背景，巨型 DOCUMENTA FIFTEEN 标题 + 插图
[Ticker 1]      红色条，滚动字幕：lumbung inter-lokal website
[Ticker 2]      紫色条，滚动字幕：ruruHaus website
[About]         绿色，正文介绍 lumbung 概念
[Lumbung 概念]  红色，"LUMBUNG" 大标题
[艺术家网格]    绿色，黑白照片卡片 + 艺术家名
[展览场地]      红色，"VENUES" 大标题
[媒体库]        黄色，三列视频卡片
[新闻]          绿色，新闻列表
[页脚]          紫色，三列链接
```

**规律：**颜色在红↔绿之间交替，黄色和紫色作为"特殊节点"出现（一个是媒体库，一个是页脚）。这种交替不是精确对称的，保留了一定的随机感。

### 布局特征

- **全宽色块**，无容器边距（padding 直接在 section 内部）
- **标题永远靠左对齐**，且尺寸极大（几乎横跨全屏宽度）
- **内容区域采用 CSS Grid**，根据内容类型切换为 2列、3列或单列
- **无任何分割线、边框、卡片阴影**——色块本身就是分区
- **图片以黑白为主**，在高饱和彩色背景上产生强烈对比
- 标题与正文的**字号落差极大**（133px 标题 vs 42px 正文），没有中间档

## 五、交互元素

### 按钮 `.button--sketched`

这是整个设计中唯一"可爱/手工感"的 UI 元素。

- 外观：椭圆形轮廓，但笔触是"手绘感"的——不是完美的 CSS border-radius，而是用 SVG 绘制的略微不规则的椭圆线条
- 字体：Söhne Regular
- 背景：透明
- 尺寸：padding 约 10px 上下 / 26px 左右
- 文字：全小写（"Read more"、"See all"、"Meet all"）——与标题的全大写形成对比

这个按钮的手绘椭圆与鼠标光标相遇，带来一种把工业界面"手工化"的感觉，和 documenta fifteen 整体的反机构姿态一致。

### Ticker（滚动字幕）

两条 ticker 分别使用不同字体（Gudea 和 Guggeli），都带有播放/暂停控制按钮（▶ ‖）——这是"播客/广播"的隐喻，将展览类比为一个持续播放的公共频道。

### 无障碍工具栏（Accessibility Bar）

导航中内置了四个访问模式切换：
- **Easy Read** — 简化语言版本
- **IS** — 图示语言（International Sign，国际手语）版本
- **Contrast** — 高对比度模式
- **Simple Design** — 简化视觉版本

这些工具不藏在"辅助功能"子菜单里，而是放在主导航的正中央，与 "Menu" 同级。这是一个强烈的声明：**无障碍访问不是附加功能，而是展览核心价值的一部分**。ruangrupa 在物理展览中也大量实践了身体多样性、语言多样性的接入实践，网站设计直接延续了这一立场。

## 六、插图风格

首页 Hero 区域有两张手绘插图（左右两侧）：
- 线条风格：黑色粗线描边，内部留白或填紫色
- 题材：手部特写、植物（热带植物/花朵）
- 风格：接近 risograph 印刷风格，或东南亚本土插画风格
- 不是精致的数字插画，而是带有"版画感"的平面图形

插图不放在内容区域，而是**溢出到 Hero 的边缘之外**，制造一种图形从画面里"长出来"的感觉，破坏了矩形版面的秩序。

## 七、总结：核心设计原则

| 原则 | 体现 |
|------|------|
| **平权而非等级** | 四色无主副，各色块无大小优劣 |
| **集体中保有个体** | MBI-DCMNT15 字体——单词内部风格破裂 |
| **机构性与手工性并置** | 巨型海报字 + 手绘椭圆按钮 + 不完美的字形 |
| **拒绝装饰** | 无渐变、无阴影、无分割线、无圆角卡片 |
| **无障碍即核心** | 访问工具栏与主导航同级 |
| **身体感/物质感** | B&W照片、手绘插图、非完美的字形曲线 |
| **对抗性的平静** | 高饱和色 + 全黑文字，强烈但不焦虑 |

---

## 八、对创作实践的启发

**如果你在做艺术家个人网站，documenta fifteen 给的启示不是"照抄它的风格"，而是：**

1. **让你的排版语言承载概念**——字体不只是信息载体，字体本身可以有立场
2. **颜色是情绪的直接材料**，不需要渐变和阴影来"表达深度"，平涂本身就有力量
3. **把"无障碍"当设计资产**，而不是后期修补
4. **允许界面里有"不完美"**——手绘按钮、混合字形，告诉访客：这不是一个企业产品
5. **设计可以有政治立场**：去中心、平权、集体，这些可以体现在版式里，不用写出来
